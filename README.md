# Webby - Evolutionary Website Builder

[繁體中文 (Traditional Chinese)](./README.zh-TW.md)

Webby is a React application built with Vite and Tailwind CSS. It functions as an "Evolutionary Website Builder" that uses the OpenRouter API (via the OpenAI SDK) to generate and iteratively refine website designs through a unique tournament-style bracket system.

## Key Concepts

*   **Tournament Bracket**: The core mechanism presents two AI-generated design choices side-by-side. Your selection advances up a binary tree bracket (`BracketNode`) until a single ultimate winner remains.
*   **Streaming AI Responses**: Experience real-time feedback. Webby uses the OpenAI SDK to stream responses from OpenRouter, rendering HTML, CSS, and JS as it's being generated by the model.
*   **Safe Iframe Previews**: All generated HTML, CSS, and JavaScript are rendered safely within `<iframe>` elements to isolate the AI-generated code from the main React application.

## Architecture

*   **`src/App.tsx`**: The main application component managing three distinct phases:
    *   `setup`: Getting the initial user prompt and enhancing it.
    *   `tournament`: Displaying candidates side-by-side and managing the binary tree tournament.
    *   `result`: Displaying the final winning design with options to copy or download code.
*   **`src/services/openrouter.ts`**: Contains the logic for interacting with the OpenRouter API. Let users specify models (e.g., `openai/gpt-5.4` or `google/gemini-3-flash-preview`). It handles:
    *   `enhancePrompt`: Expands the initial user idea into a detailed prompt.
    *   `generateCandidateStream`: Generates the initial design candidates (HTML, CSS, JS, and design philosophy).
*   **`src/i18n.ts`**: Handles internationalization/translations for the application UI.

## Getting Started

### Prerequisites

*   Node.js
*   An [OpenRouter API](https://openrouter.ai/) Key

### Setup & Installation

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Environment Variables**:
    Create a `.env.local` file in the root directory and add your OpenRouter API key:
    ```env
    OPENROUTER_API_KEY=your_openrouter_api_key_here
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

### Available Scripts

*   `npm run dev`: Start local development server.
*   `npm run build`: Build for production.
*   `npm run preview`: Preview production build locally.
*   `npm run lint`: Type check / Lint (runs `tsc --noEmit`).
*   `npm run clean`: Clean build artifacts.
